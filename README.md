A mock server for client-server testing purpose.

## Configuration
<pre>
"require": {
        "micronative/mock-server": "^1.0.0"
},
"repositories": [
    { "type": "vcs", "url": "https://github.com/micronative/mock-server" }
],
</pre>

Run
<pre>
composer require micronative/mock-server:1.0.0
</pre>
